# Causes-of-Death-Worlwide (Year 2021)
Analysis of the top ten causes of death worldwide using MySQL Workbench.

Data downloaded from https://data.who.int/countries/ as CSV file.

Correlation of country codes and country name exported as CSV file from https://apps.who.int/gho/data/node.searo-metadata.COUNTRY?lang=en 

**Raw data**
- Causes of deaths for all countries: https://mardedexo.github.io/Causes-of-Death-Worlwide/RawData/ORIGINAL_GHE_FULL_DD.csv
- Countries-code correlation: https://mardedexo.github.io/Causes-of-Death-Worlwide/RawData/CodeList.csv 

### Data cleaning and process:

**Database creation script code:** https://mardedexo.github.io/Causes-of-Death-Worlwide/causesofdeathworldwide_dbCreation.sql

**Analysis code:** https://mardedexo.github.io/Causes-of-Death-Worlwide/Death_causes_worldwide_top10_SQLanalysis.sql

**Data cleaned:** https://mardedexo.github.io/Causes-of-Death-Worlwide/Top10ByCountry_all_fullNameCleanedxlsx
